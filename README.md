# P2HACKS2025 アピールシート

## プロダクト名
Memet(メメット)

## コンセプト
思い出は、残すものではなく、みんなで思い出すもの。


## 対象ユーザ
- 家族、友人、仲間と「一緒に過ごした時間」を持っている人
- 修学旅行・合宿・旅行・ハッカソンなど、共有の思い出があるグループ
- 写真はたくさん撮っているが、見返す機会がない
- ひとりでスマホを眺めるだけになっている
- 思い出を「記録」よりも「体験」として大切にしたい人

## 利用の流れ
### 思い出を残すとき
1. NFCチップ入りのカードをかざしてアプリを起動
2. 思い出のタイトルを入力
3. 限られた枚数の写真を撮影
4. 写真とメッセージを保存
### 思い出を振り返るとき
1. 参加者それぞれがカードを持ち寄る
2. カードの裏を組み合わせて「アルバムを開くカギ」を作る
3. 全員がそろったときだけ、アルバムが開く
4. 同じ画面を囲みながら、思い出を振り返る


## 推しポイント
- ひとりでは開けないアルバム  
 → みんなが集まらないと思い出せない設計

- 写真の量をあえて制限  
 → 1枚1枚に意味が残り、記憶に残る

- カードというモノの存在  
 → スマホの中に埋もれない、思い出の入口になる

- 感情と空気を思い出す体験  
 → 写真を見るだけでなく、その場の会話が自然に生まれる
